# irtelemetry
Barebones NodeJS iRacing telemetry project using [node-irsdk-2023](https://github.com/hmiefert/node-irsdk-2023).

# get it up and running
```
git clone https://github.com/hmiefert/irtelemetry
cd irtelemetry
npm install
npm run start
```
